CJD LOGIC/BOARD

A complete, self-contained digital-logic workbench that runs entirely in your browser. Wire up gates, adders, multiplexers and flip-flops, watch current flow in real time, and read out the Boolean equation as you build.

Open the Simulator Download the App

Features

Everything you need to design, simulate, and understand a digital circuit — no installs, no accounts, no server.

Drag & Drop Design

Drag parts from the palette, move them freely, rotate them, delete them — the whole board responds instantly.

Pin-to-Pin Wiring

Drag from an output pin to an input pin to connect. Wires glow and animate while carrying a HIGH signal.

Boolean Equations

Every LED output shows its live symbolic Boolean expression, traced automatically through the circuit.

Digital Circuits

Half/Full Adders, a Multiplexer, Demultiplexer, Encoder, Decoder, and Comparator as ready-made ICs.

Sequential Logic

SR/JK/D/T flip-flops, a 4-bit Register, and a 4-bit Counter — all clocked, all stateful.

Truth Tables

Generate a full truth table for your circuit, or right-click any single part for its own datasheet.

Save / Open / Export

Save a project as JSON, reopen it later, or export a PNG snapshot to drop into a report or slide deck.

Zoom, Pan & Minimap

Scroll to zoom, drag empty space to pan, and use the minimap to jump around large circuits.

Examples

One click loads a working circuit straight into the Live Simulator above.

Half Adder

Two switches feed a Half Adder IC, showing Sum and Carry on two LEDs.

SR Latch

A manual clock drives an SR flip-flop; watch Q and Q̄ hold their state between pulses.

4-bit Counter

An auto-running clock drives a 4-bit binary counter across four LEDs.

2:1 Multiplexer

Two data switches and a select switch route one signal through to a single LED.

Download

CJD LOGIC/BOARD is a single HTML file with no server and no dependencies beyond one CDN script used only for PNG export. Save it once and it runs offline forever.
